# ============================================================================
# Installation Instructions
# ============================================================================
#
# 1. Create a virtual environment (recommended):
#    python -m venv venv
#    
#    Activate it:
#    - Windows: venv\Scripts\activate
#    - Mac/Linux: source venv/bin/activate
#
# 2. Install dependencies:
#    pip install -r requirements.txt
#
# 3. Install Tesseract OCR (for scanned PDFs):
#    - Windows: https://github.com/UB-Mannheim/tesseract/wiki
#    - Mac: brew install tesseract
#    - Linux: sudo apt-get install tesseract-ocr
#
# 4. Install poppler (required for pdf2image):
#    - Windows: Download from https://github.com/oschwartz10612/poppler-windows/releases/
#              Add to PATH or set poppler_path in code
#    - Mac: brew install poppler
#    - Linux: sudo apt-get install poppler-utils
#
# 5. For GPU acceleration (optional but recommended):
#    - Install CUDA Toolkit: https://developer.nvidia.com/cuda-downloads
#    - Reinstall PyTorch with CUDA:
#      pip install torch==2.1.2 --index-url https://download.pytorch.org/whl/cu118
#
# 6. Hugging Face Model Access:
#    - Create account at https://huggingface.co/
#    - For LLaMA models, request access at: https://huggingface.co/meta-llama/Llama-3.1-8B-Instruct
#    - Login via CLI: huggingface-cli login
#    - Enter your access token when prompted
#
